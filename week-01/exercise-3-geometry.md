## **EXERCISE 3**

Create a Java application named `GeometryCalculator` that computes area and perimeter values for three shapes using Java's `Math` class.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Declare the following variables: `circleRadius = 5.5`, `rectangleLength = 8.2`, `rectangleWidth = 3.7`, `triangleBase = 6.0`, `triangleHeight = 4.5`
- Calculate circle area using `Math.PI` and `Math.pow()` to square the radius
- Calculate circle circumference using `Math.PI`
- Calculate rectangle area and perimeter using basic arithmetic operators
- Calculate triangle area, then use `Math.sqrt()` to find the hypotenuse (assume a right triangle)
- Round all results using `Math.round()` or cast to a reasonable number of decimal places
- Print labeled output for each shape showing its dimensions and calculated values, plus a summary line with the total area across all three shapes
