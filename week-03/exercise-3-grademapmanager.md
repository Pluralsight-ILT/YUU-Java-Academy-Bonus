# **Exercises: Grade Book Map**

Build a menu-driven grade book using `HashMap<String, Double>` to store student names and their grades.

Remember to create your projects in the pluralsight/workbook-3 folder.

---

## **EXERCISE 3**

Create a Java application named `GradeBookMap`.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Create a `HashMap<String, Double>` to store names and grades
- Implement methods: `addGrade()`, `updateGrade()`, `removeStudent()`, `getGrade()`, `displayAllGrades()`
- Use `put()`, `get()`, `remove()`, `containsKey()`, `keySet()` where appropriate
- Calculate and display the class average
- Handle lookups for names that don't exist in the map
- Use Scanner and a menu to drive the program
