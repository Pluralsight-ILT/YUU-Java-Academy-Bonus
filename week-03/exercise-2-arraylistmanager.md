# **Exercises: ArrayList Manager**

Build a menu-driven app that manages a list of student names using `ArrayList<String>`.

Remember to create your projects in the pluralsight/workbook-3 folder.

---

## **EXERCISE 2**

Create a Java application named `ArrayListManager`.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Create an `ArrayList<String>` to store student names
- Implement a menu with methods: `addStudent()`, `removeStudent()`, `searchStudent()`, `displayAllStudents()`
- Handle the case where a name is not found
- Use Scanner for all user input
