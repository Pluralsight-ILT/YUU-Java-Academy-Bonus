# **Exercises: Text Analyzer**

Given a sentence or paragraph entered by the user, analyze it with string methods and print a stats report.

Remember to create your projects in the pluralsight/workbook-2 folder.

---

## **EXERCISE 3**

Create a Java application named `TextAnalyzer`.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Use Scanner to collect a sentence or paragraph from the user
- Count total characters, characters without spaces, and estimate word count
- Find the first and last position of a specific character using `indexOf()` and `lastIndexOf()`
- Check whether specific words are present using `contains()`
- Extract the first word, last word, and a middle section using `substring()`
- Prompt the user to enter a search word and report its position in the text
- Print a report with all findings
