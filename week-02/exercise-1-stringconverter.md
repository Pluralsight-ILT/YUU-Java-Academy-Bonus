# **Exercises: String Converter**

Take a string entered by the user and apply a series of string methods. Display the result of each transformation and at least one example where methods are chained together.

Remember to create your projects in the pluralsight/workbook-2 folder.

---

## **EXERCISE 1**

Create a Java application named `StringConverter`.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Use Scanner to prompt the user for any text string
- Apply and display results for: `trim()`, `toUpperCase()`, `toLowerCase()`, `replace()`, `substring()`
- Show the original string length vs trimmed string length
- Let the user choose what word to replace when using `replace()`
- Extract at least two different substrings using different index parameters
- Chain at least two methods together and display the result
- Display before/after for each transformation
