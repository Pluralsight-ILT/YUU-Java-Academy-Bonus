# **Exercises: String Type Converter**

Read three values from the user as strings, convert them to their actual types, perform calculations, then convert the results back to strings.

Remember to create your projects in the pluralsight/workbook-2 folder.

---

## **EXERCISE 2**

Create a Java application named `StringTypeConverter`.

**Remember:** When creating a new Java project, create a new git repository and **commit your changes often!** Don't forget to push to github.com. Always create the Github repository before creating a new Java project.

**Requirements:**
- Use Scanner to collect three inputs: one representing an integer, one a decimal, one `true` or `false`
- Convert each using `Integer.parseInt()`, `Double.parseDouble()`, `Boolean.parseBoolean()`
- Perform at least one calculation with each converted numeric value
- Convert results back to strings using `String.valueOf()`
- Display each original string input alongside its converted value
- Display the results of your calculations
